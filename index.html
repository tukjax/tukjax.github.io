hexo相关

Posted on 2018-05-29 | In 杂技
  1. GIT自带的命令生成SSH key ssh-keygen 默认是RSA算法，在GIT上添加上公钥
  2. 创建 用户名.github.io的repository
  3. 因为在国内，先把NPM改成了TAOBAO提供的镜像，然后才用NPM安装了HEXO，当然NODEJS之前已经装好了
  4. hexo安装 npm install安装所需组件（这步也许不是必须的？好像没有LOG）hexo init hexo s –debug 本地端口4000 在config.yml里设置远程仓库的地址，到这里基本就算完了

目前我更新文章hexo new “” hexo generate hexo deploy 这很繁琐，这两天我写个脚本简化一下这个过程，现在我累了
我现在用的NEXT主题是CLONE下来放到HEXO的THEMES下就可以用了，当然之前要改HEXO根目录config.yml里的THEME
其余就是主题美化的部分了
